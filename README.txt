# Garcia Fretes & Mudanças — Landing Page (GitHub Pages)

## Como publicar no GitHub Pages
1. Abra o repositório: `mgtour360graus/Garcia-Fretes-e-mudanca`
2. Faça upload destes arquivos na raiz:
   - `index.html`
   - `styles.css`
   - `script.js`
   - pasta `assets/`
3. Em **Settings → Pages**, selecione:
   - Source: `Deploy from a branch`
   - Branch: `main` / folder `/ (root)`
4. Acesse:
   https://mgtour360graus.github.io/Garcia-Fretes-e-mudanca/
