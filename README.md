# Landing Page — Garcia Fretes & Mudanças

Arquivos prontos para hospedar no GitHub Pages.

## Como publicar (GitHub Pages)
1. Abra o repositório `Garcia-Fretes-e-mudanca`
2. Envie **todos os arquivos** desta pasta para a raiz do repositório:
   - `index.html`
   - `styles.css`
   - `script.js`
   - pasta `assets/`
3. Em **Settings → Pages**:
   - Source: **Deploy from a branch**
   - Branch: **main** / folder: **/** (root)
4. Aguarde a publicação e acesse:
   https://mgtour360graus.github.io/Garcia-Fretes-e-mudanca/
